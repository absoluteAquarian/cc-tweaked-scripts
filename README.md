# cc-tweaked-scripts
A collection of Lua scripts for the Minecraft mod, CC: Tweaked.

The general-purpose installer for this repository's scripts can be executed using the following in a Computer's shell:
```
wget https://raw.githubusercontent.com/absoluteAquarian/cc-tweaked-scripts/main/installer.lua installer
```
The `rom` folder and `bios.lua` file are manually copied from the [CC:Tweaked source](https://github.com/cc-tweaked/CC-Tweaked/tree/mc-1.20.x/projects/core/src/main/resources/data/computercraft/lua) directory.
